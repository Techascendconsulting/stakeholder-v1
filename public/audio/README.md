# Motivation Audio Files

This directory should contain the following MP3 files for the Motivation Page:

## Required Files:
- `daily.mp3` - Today's 5-Minute Talk (5:00)
- `overcoming-overwhelm.mp3` - Overcoming Overwhelm (7:00)
- `discipline.mp3` - The Discipline Dividend (4:40)
- `salary.mp3` - Your Next Salary is Waiting (5:20)
- `rejection.mp3` - Resilience in Rejection (4:20)

## Getting Audio Files:

### Option 1: Record Your Own
- Record 5-minute motivational talks
- Export as MP3 format
- Place in this directory

### Option 2: Use Text-to-Speech
- Use ElevenLabs or similar service
- Generate audio from motivational scripts
- Export as MP3

### Option 3: Use Free Audio
- Find royalty-free motivational content
- Ensure proper licensing for commercial use
- Convert to MP3 if needed

## File Requirements:
- Format: MP3
- Quality: 128kbps or higher
- Duration: 4-6 minutes each
- Content: Motivational/inspirational talks

## Note:
The Motivation Page will show "Audio not available" if files are missing, but the interface will still work for future audio integration.
